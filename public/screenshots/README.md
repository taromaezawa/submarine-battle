# submarine-battle Screenshots

These are visual representations of the submarine-battle game interface:

- **room-join.svg** - Initial screen where players enter a 4-digit room code and their name
- **fleet-deployment.svg** - Grid-based interface for placing ships (submarine, destroyer, cruiser)
- **game-board.svg** - Active gameplay showing both your fleet and enemy grid with hits/misses

These SVG files display correctly in browsers and GitHub's web interface.
