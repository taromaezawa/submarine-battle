# Contributing to submarine-battle

We welcome pull requests! Here's how to contribute.

## Getting Started

1. Fork and clone: `git clone https://github.com/YOUR_USERNAME/submarine-battle`
2. Install: `npm install`
3. Start: `npm start`
4. Create a branch: `git checkout -b feature/new-feature`

## Development

### Code Style

- **JavaScript**: ES6+, no TypeScript required
- **Formatting**: Use Prettier if available
- **Testing**: Add tests for game logic

### Testing

```bash
npm test
```

Tests cover:
- Ship placement validation
- Hit/miss detection
- Win condition

## Pull Request Process

1. Update README.md if needed
2. Add/update tests
3. Ensure `npm test` passes
4. Create a PR with clear description
5. Wait for review

## Reporting Issues

Open a GitHub issue with:
- Browser/OS
- Steps to reproduce
- Expected vs. actual behavior
